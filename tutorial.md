## GIT 명령어 요약

- clone : 원격 저장소 복사
- add : 스테이지 영역에 작업 파일을 추가
- commit : 코드 데이터를 세이브, 스테이지 영역의 파일들을 가지고 커밋(세이브)를 만들 수 있다.
- push : 원격 저장소에 업로드

## 브랜치 변경하기

- 브랜치(branch) : 기존 내용을 유지한 채 새로운 내용을 추가하고 싶을 때 사용
- 체크아웃 : 특정 브랜치(혹은 커밋)으로 돌아갈 때 사용
- 소스트리 체크아웃 : 브랜치 이름을 더블클릭하는 것으로 체크아웃 가능
